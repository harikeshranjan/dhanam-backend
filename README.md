# Dhanam - Personal Finance Management Backend

Welcome to the **Dhanam Backend**, a lightweight and production-ready backend built with **Express.js** and **TypeScript**. This system is designed to power a single-user personal expense and budget management application.

The backend employs a **token-based (Bearer) authentication model**, making it ideal for both mobile apps (iOS) and web clients. It avoids reliance on cookies or browser sessions, ensuring simplicity, security, and flexibility. This design allows the backend to be reused seamlessly across multiple platforms.
